# RAHHH

Create a `.env` file in the project root and add your Financial Modeling Prep API key:

```env
FMP_API_KEY=your_key_here
```

Run the project with `python main.py` (or `./.venv/bin/python main.py` if you use the local virtual environment), or launch `Python: Debug main.py` from VS Code Run and Debug. The VS Code launch configuration already loads `.env`.
